## Description

Provides an API to the ruby-advisory-db. Handles updating and loading the
Database.

## Features

* Downloads the [ruby-advisory-db] to `~/.gem/advisories`.
* Handles updating the Database.

## Examples

    require 'rubygems/advisories'
